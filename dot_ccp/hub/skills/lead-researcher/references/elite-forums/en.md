## Key Landscape Insights

| Insight | Detail |
|---------|--------|
| Post-Reddit migration | Serious devs moved to Lemmy/programming.dev, fosstodon, infosec.exchange, Tildes — not back to Reddit |
| Quality gate proxies | Payment barrier (WIP.co) or revenue threshold (MicroConf $10K+ MRR) = strongest signal quality indicators |
| Fediverse not indexed | fosstodon.org, infosec.exchange = NOT searchable by Google — require direct access, but where security researchers went post-Twitter |
| 0x00sec relaunch | Fresh Discourse instance January 2026, clean moderation, emphasis on explaining "why" not just results |
| RE community | 0x00sec (general hacking) + Tuts4You (RE since 2003) + HackTheBox (intermediate practitioners) = three distinct roles |
| ML community | MLOps Community (production) + DataTalks.Club (structured learning) + Hugging Face (model-level). No single academic ML hub. |
| Lobste.rs milestone | Passed 20,000 users April 2026 post-Reddit exodus. Invite-only keeps SNR high. |
| Game dev CTFtime note | CTFtime.org is a calendar/tracker not a forum — per-CTF discussion happens on Discord |
