## Key Landscape Insights

| Insight | Detail |
|---------|--------|
| GeekNews is #1 | Undisputed top KR tech aggregator — 5K+ companies use it as internal tech feed |
| Security triad | Dreamhack (training platform) + webhacking.kr (wargames) + CODEGATE (elite competition) = distinct complementary roles |
| IAMROOT | Most elite pure-technical KR community — closest Korean equivalent to a graduate-level systems programming seminar |
| Dead communities | KLDP (historic Linux) appears defunct (connection refused). DevPia (.NET) dead (TLS expired). DC Inside programming gallery degraded. |
| Platform fragmentation | Forums (OKKY) + blogs (Velog) + career SNS (Careerly) + news (GeekNews) + Discord/Slack (language groups) + Naver Cafes (topic guilds) |
| Hardware gap | No dedicated elite KR hardware/maker/IoT forum. Scattered across Naver Cafes and Facebook groups. |
| Disquiet role | Korea's Indie Hackers — primary platform for makers and startup founders. Very different demographic from OKKY. |
| Content farm severity | Naver search heavily polluted with AI-generated developer content. Tistory and Naver blog = distrust by default. |
