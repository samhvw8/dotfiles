## Key Landscape Insights

| Insight | Detail |
|---------|--------|
| 1C ecosystem | Infostart (1.5M specialists) is uniquely Russian/CIS — 1C:Enterprise dominates CIS ERP/accounting. Zero Western parallel. Invisible to EN research. |
| ODS.AI history | Original Slack (60K+ members) suspended by Slack in 2023 — community migrated to own platform. DataFest 2026 confirmed. |
| Security bifurcation | Codeby.net (legitimate, 150K specialists) vs underground forums (xss.is admin arrested). Codeby is the research-safe target. |
| LOR vs OpenNET | linux.org.ru = larger, community-oriented. OpenNET = older (1996), documentation/news-focused. Both complementary. |
| Fragmentation to Telegram | Many former forums (Vingrad dead) now exist as Telegram chats. Hexlet/ru-local-communities GitHub catalogs 100+ city-specific TG dev communities. |
| Free readability | Most RU forums freely readable without registration: wasm.in, codeby.net, habr.com, linux.org.ru, opennet.ru, overclockers.ru. |
| Startup community | Mesto.co = paywalled startup/founder community (HTTP 402). Product Radar = open Product Hunt equivalent. |
| RadioKot/KAZUS | RadioKot (maker/Arduino) + KAZUS (professional EE/embedded) form the hardware specialist tier below Overclockers. |
