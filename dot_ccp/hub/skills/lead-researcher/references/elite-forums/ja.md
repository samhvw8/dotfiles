## Key Landscape Insights

| Insight | Detail |
|---------|--------|
| No JP HackerNews | Closest combo: Zenn (quality writing) + はてなブックマーク (link aggregation with commentary). No voting-based link community exists. |
| connpass as alive-proxy | Almost every active JP community organizes events on connpass — activity there = best proxy for which communities are actually alive |
| AtCoder uniqueness | Only platform globally where competitive programming ratings are directly used by employers for hiring — tight feedback loop |
| Zenn vs Qiita gap | Widened noticeably since 2023. Serious engineers publish on Zenn first; Qiita increasingly beginner/SEO platform. |
| fabcross closed | Japan's premier maker/IoT media community closed permanently March 31, 2025 — significant gap, no clear successor |
| teratail active | Added Talk Room β June 2026, updated terms to prohibit AI training data scraping — actively expanding |
| 5ch プログラム板 | Only fully anonymous JP dev space — valuable for unfiltered expert opinions unavailable elsewhere. NOT Google-indexed. |
| Security ecosystem | Three tiers: SECCON (elite competitions) + IPA Security Camp (elite youth training) + JPCERT/ISOG-J (professional orgs). Complete ecosystem. |
| Slack dominance | Slack preferred over Discord in JP professional communities. Free Slack's 90-day message retention drives Discord migration for younger communities. |
| JP ML fragmentation | Team AI (10K+ Facebook+Slack) + Kaggle Japan (Slack) + research Discords — no single ML hub |
