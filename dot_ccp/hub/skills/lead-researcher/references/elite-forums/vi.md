## Key Landscape Insights

| Insight | Detail |
|---------|--------|
| Two-tier structure | Elite tier: Viblo (written articles, curated) + VOZ/Dạy Nhau Học (forum discussion). Everything else is lower signal. |
| Viblo ecosystem | Most developed: Blog + Code (competitive programming) + CTF + Learning + Battle + Game sub-platforms |
| VOZ registration | Requires CCCD (Vietnamese national ID) since 2024 — filters bots, guarantees Vietnamese authenticity |
| HVA → WhiteHat | Vietnam's original hacker community HVA (founded 1997) closed 2014 → spawned WhiteHat.vn as institutional successor |
| Dead platforms | SinhVienIT.net closed August 2019. kipalog.com + congdongcviet.com both redirect to unrelated Japanese company — dead. |
| Content theft scandal | TopDev/TechTalk/TechBlog (all Applancer) dominate Google results but carry stolen content — high visibility, zero original signal |
| AI/ML fragmentation | No dedicated standalone ML forum of quality. Community lives in Facebook groups (vnaiforum, congdongaimlvietnam). |
| CTF strength | Vietnam top-10 globally in CTF participation. Teams Nine Dragons/CLGT, Bamb00-vn, PiggyBird placed top-10 internationally. |
| Indie gap | No native indie hacker/solopreneur forum. Movement lives on Threads/Facebook. IndieHackers.com/post/indie-hackers-in-vietnam is primary gathering. |
