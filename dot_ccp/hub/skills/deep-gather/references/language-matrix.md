## UNIQUE Decision Test

Before adding a language, score (YES=1, NO=0). Add if >= 3.

```
U - Unique: Non-duplicative insights (not EN translations)?
N - Non-trivial: >1000 relevant articles/discussions?
I - Indexed: Search engines can access?
Q - Quality: Platforms with quality enforcement?
E - Expert: Domain experts publish in this language?
```
